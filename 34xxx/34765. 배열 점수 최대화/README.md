# [Gold IV] 배열 점수 최대화

[문제 링크](https://www.acmicpc.net/problem/34765) 

## 문제 설명

<p>초깃값이 양의 정수 $K$인 변수 $V$가 있다. 쿠는 초기에 원소가 없는 배열 $A$에 아래의 연산을 총 $N$번 수행하려고 한다.</p>

<ul>
<li>$1$번 행동과 $2$번 행동 중 한 가지를 골라서 수행한다.
<ul>
<li>$1$번 행동은 배열 $A$ 뒤에 $V$를 추가한 이후, $V$를 $1$만큼 감소시킨다.</li>
<li>$2$번 행동은 $V$를 $1$만큼 증가시킨다.</li>
</ul>
</li>
</ul>

<p>$1$번 행동은 반드시 한 번 이상 수행해야 하며, 연산 중 $V$가 음수가 될 수 있음에 유의하라.</p>

<p>배열 $A$의 점수란 $A$의 길이와 $A$의 중앙값의 곱이다. 중앙값이란 $A$의 길이를 $M$이라고 할 때, $A$를 오름차순으로 정렬했을 때의 $\lceil$$\frac{M}{2}$$\rceil$번째 원소이다. 예를 들어 $[1, 2, 1]$의 중앙값은 $1$이고, $[2, 4, 3, 1]$의 중앙값은 $2$이다.</p>

<p>$N$번의 연산을 마친 후, 배열 $A$의 점수가 최대가 되도록 하고 싶다. 연산을 최적으로 할 때 얻을 수 있는 배열의 점수의 최댓값을 구해보자.</p>

## 입력 

 <p>연산의 횟수, 변수 $V$의 초깃값을 의미하는 정수 $N$, $K$가 공백으로 구분되어 주어진다. $(1\leq N, K\leq 100\, 000)$</p>

## 출력 

 <p>연산을 최적으로 할 때 얻을 수 있는 배열의 점수의 최댓값을 출력한다.</p>

# [Platinum IV] 숫자 배치하기

[문제 링크](https://www.acmicpc.net/problem/34768) 

## 문제 설명

<p>쿠는 아래의 조건을 만족하는 $N\times N$ 행렬을 찾고 싶어졌다. 이때 $N$은 짝수이다.</p>

<ul>
<li>$1 \le p \le \frac{N^2}{2}$인 모든 양의 정수 $p$에 대해 아래의 조건을 만족한다.
<ul>
<li>행렬에 $p$가 정확히 $2$번 등장한다.</li>
<li>$p$의 위치를 각각 $(r_1, c_1) , (r_2, c_2)$라 하자. 이때 $|r_1 - r_2| + |c_1 - c_2| > 1$이고, $(r_1, c_1) , (r_2, c_2)$를 포함하는 가장 작은 부분 행렬에 속한 값들의 합은 $p$의 배수이다.</li>
</ul>
</li>
</ul>

<p>쿠를 위해 조건을 만족하는 행렬을 아무거나 하나 출력해 보자. 조건을 만족하는 행렬이 하나 이상 존재함이 보장된다.</p>

## 입력 

 <p>짝수인 양의 정수 $N$이 주어진다. $(2 \le N \le 1\,000)$</p>

## 출력 

 <p>$N$개의 줄에 걸쳐, 조건을 만족하는 행렬의 각 행에 속하는 $N$개의 값을 공백으로 구분하여 출력한다.</p>

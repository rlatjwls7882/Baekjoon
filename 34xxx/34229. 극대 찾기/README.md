# [Diamond III] 극대 찾기

[문제 링크](https://www.acmicpc.net/problem/34229) 

## 문제 설명

<p>크기 $N \times N$의 숨겨진 배열 $A$가 있다. 배열 $A$의 원소는 $1$부터 $N^2$사이의 서로 다른 정수로 이루어져 있다.</p>

<p>다음 조건을 만족하는 $(x,y)$를 극대라 한다.</p>

<ul>
	<li>$x>1$인 경우, $A_{x,y} > A_{x-1,y}$를 만족한다.</li>
	<li>$x<N$인 경우, $A_{x,y} > A_{x+1,y}$를 만족한다.</li>
	<li>$y>1$인 경우, $A_{x,y} > A_{x,y-1}$를 만족한다.</li>
	<li>$y<N$인 경우, $A_{x,y} > A_{x,y+1}$를 만족한다.</li>
</ul>

<p>당신은 배열 $A$에서 극대를 찾아야 한다.</p>

<p>답을 찾기 위해, 당신은 채점 시스템에 두 종류의 연산을 최대 $27$회 할 수 있다:</p>

<ul>
	<li><code>V</code> $i$ $j$ $k$: $\max(A_{i,j}, A_{i+1,j}, \ldots, A_{i+k-1,j})$의 값을 묻는다.</li>
	<li><code>H</code> $i$ $j$ $k$: $\max(A_{i,j}, A_{i,j+1}, \ldots, A_{i,j+k-1})$의 값을 묻는다.</li>
</ul>

<p>극대 $(x,y)$를 찾아보자. 만약 그러한 답이 여러 가지 있다면, 아무 답이나 찾아보자.</p>

## 입력 

 <p>첫째 줄에 $N$이 주어진다. ($2 \leq N \leq 2\,000$)</p>

<p>이후 당신과 채점 시스템과의 인터랙션이 진행된다.</p>

## 출력 

 Empty

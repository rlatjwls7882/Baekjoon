# [Gold IV] 땅따먹기

[문제 링크](https://www.acmicpc.net/problem/34763) 

## 문제 설명

<p>무한히 넓은 좌표평면이 있다. 초기에는 $(0,0)$에 $0$이 적혀있고, 나머지 정수 좌표에는 아무 값도 적혀 있지 않다.</p>

<p>당신은 큐를 이용해 아래의 행동을 $N$번 수행하려고 한다. 초기에 큐는 비어 있다.</p>

<ol>
<li>$0$이 적혀 있는 좌표 중 $1$개를 골라서, 해당 좌표를 큐에 넣는다.</li>
<li>아래의 행동을 차례대로 수행한다.
<ul>
<li>큐의 맨 앞에서 좌표 $1$개를 꺼내고, 꺼낸 좌표에 $1$을 적는다.</li>
<li>꺼낸 좌표와 상하좌우로 인접한 정수 좌표 중, $0$이 적혀 있는 모든 정수 좌표를 큐의 맨 뒤에 넣는다. 큐에 좌표를 넣는 순서는 결과에 영향을 주지 않는다.</li>
<li>꺼낸 좌표와 상하좌우로 인접한 정수 좌표 중, 아무 값도 적혀 있지 않은 모든 좌표에 $0$을 적는다.</li>
</ul>
</li>
<li>큐가 빌 때까지 $2.$을 반복한다.</li>
</ol>

<p>행동을 어떤 방식으로 수행하더라도, 한 좌표는 큐에 최대 한 번만 들어감을 증명할 수 있다.</p>

<p>행동을 $N$번 수행했을 때, $1$이 적힌 좌표의 개수가 정확히 $K$가 될 수 있는지 판별해 보자.</p>

## 입력 

 <p>첫 번째 줄에 테스트 케이스의 수를 의미하는 정수 $T$가 주어진다. $(1\leq T\leq 100)$</p>

<p>각 테스트 케이스마다 행동의 횟수를 의미하는 정수 $N$, $1$이 적힌 좌표 개수의 목표를 의미하는 정수 $K$가 공백으로 구분되어 주어진다. $(1\leq N, K\leq 10^9)$</p>

## 출력 

 <p>각 테스트 케이스마다 $1$이 적힌 좌표의 개수가 정확히 $K$가 될 수 있다면 <span style="color:#e74c3c;"><code>YES</code></span>를, 그렇지 않다면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>

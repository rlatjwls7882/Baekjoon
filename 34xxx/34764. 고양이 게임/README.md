# [Gold IV] 고양이 게임

[문제 링크](https://www.acmicpc.net/problem/34764) 

## 문제 설명

<p>양의 정수 $A$가 주어질 때, 건덕이와 건구스는 이를 이용해 고양이 게임을 하려 한다. 게임판은 $2A+1$개의 칸으로 이루어져 있고, 모든 칸은 가로로 일렬로 이어져 있다. 처음에 정중앙인 $A+1$번째 칸에 고양이 로봇이 있다.</p>

<p>건덕이와 건구스가 차례대로 아래의 행동을 반복한다.</p>

<ul>
<li>건덕이는 고양이 로봇을 왼쪽이나 오른쪽으로 $1$칸 이동시킨다.</li>
<li>건구스는 고양이 로봇을 왼쪽이나 오른쪽으로 $2$칸 이동시킨다.</li>
</ul>

<p>고양이 로봇을 게임판 밖으로 이동시킬 수 없고, 한 번 놓였던 칸으로도 이동시킬 수 없다. 시작 칸인 $A+1$번째 칸은 고양이 로봇이 놓였던 칸으로 생각한다.</p>

<p>둘 다 최선을 다해서 승부를 겨루며, 만약 고양이 로봇을 이동시킬 수 없다면 해당 플레이어가 패배하고, 상대방이 승리한다.</p>

<p>양의 정수 $A$가 주어질 때, 건덕이와 건구스 중 누가 승리하는지 구해보자.</p>

## 입력 

 <p>정수 $A$가 주어진다. $(1\leq A\leq 10^9)$</p>

## 출력 

 <p>건덕이가 이긴다면 <span style="color:#e74c3c;"><code>duck</code></span>을, 건구스가 이긴다면 <span style="color:#e74c3c;"><code>goose</code></span>를 출력한다.</p>

# [Silver V] 수열과 수열 2

[문제 링크](https://www.acmicpc.net/problem/34155) 

## 문제 설명

<p><strong>길이 </strong>$N$<strong>의 수열</strong> $A = \left(A_1, \, A_2 , \, \cdots, \, A_N\right)$이 주어진다. 이때, 다음을 만족하는 <strong>함수</strong> $f$의 개수를 구하라.</p>

<ul>
	<li>$f : \left\{1,  \, 2, \, \cdots, \, N  \right\} \rightarrow \left\{1,  \, 2, \, \cdots, \, N  \right\}$. 즉, $f$는 정의역과 공역으로 집합 $\left\{1,  \, 2, \, \cdots, \, N  \right\}$을 가진다.</li>
	<li>$1 \leq i \leq N$을 만족하는 모든 정수 $i$에 대해, $f(i) \neq i$이고 $f(i) \neq A_i$이다.</li>
</ul>

<p>이때, 답이 매우 커질 수 있으므로 $998 \, 244 \, 353$으로 나눈 나머지를 출력하라. $998 \, 244 \, 353$은 소수이다. </p>

## 입력 

 <p>첫 번째 줄에 수열의 길이 $N$이 주어진다. $(2 \leq N \leq 200 \, 000)$</p>

<p>두 번째 줄에 수열 $A$의 원소 $A_1, \, A_2, \, \cdots, \, A_N$이 공백으로 구분되어 주어진다. ($1 \leq A_i \leq N$)</p>

## 출력 

 <p>조건을 만족하는 함수 $f$의 개수를 $998 \, 244 \, 353$으로 나눈 나머지를 구하라.</p>

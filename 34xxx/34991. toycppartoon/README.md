# [Silver III] toycppartoon

[문제 링크](https://www.acmicpc.net/problem/34991) 

## 문제 설명

<p>토카는 새로운 언어를 공부하기 위해 새로운 계정을 만들기로 했다. 하지만 토카는 계정의 이름을 정하지 못하고 있다!</p>

<p>토카가 이번에 공부할 언어의 이름은 $S$이다. 토카는 원래 사용하고 있던 계정 이름인 <span style="color:#e74c3c;"><code>toycartoon</code></span>과 비슷하게 이름을 짓기로 했다.</p>

<p>하지만 평소 쓰던 계정 이름에 공부할 언어의 이름만 뒤에 붙이는 것은 식상하다고 느낀 토카는 다음과 같은 방식으로 새로운 <strong>계정 이름의 후보</strong>를 정하기로 했다.</p>

<ul>
<li>$S$의 접두사이면서 <span style="color:#e74c3c;"><code>toycartoon</code></span>의 부분문자열인 것 중 가장 긴 문자열을 $O$라고 하자. 그러한 문자열이 여러 개라면 가장 앞에 있는 문자열을 선택한다. 

<ol>
<li>만약 $O$의 길이가 0이라면 <span style="color:#e74c3c;"><code>toycartoon_</code></span> 뒤에 $S$를 붙인 문자열이 <strong>계정 이름의 후보</strong>가 된다.</li>
<li>만약 $O$의 길이가 1 이상이라면 <span style="color:#e74c3c;"><code>toycartoon</code></span>은 세 문자열 $T$, $O$, $Y$를 순서대로 쓴 것으로 볼 수 있다. $S$에서 $O$를 제외한 부분의 접미사이면서 $Y$의 접두사인 가장 긴 문자열을 $Y$의 앞에서 지운다. 이후 $T$, $S$, $Y$를 이어붙인 문자열이 <strong>계정 이름의 후보</strong>가 된다.
<ul>
<li>예를 들어 $S$가 <a href="https://www.rust-lang.org/"><span style="color:#e74c3c;"><code>rust</code></span></a>라면 $O$는 <span style="color:#e74c3c;"><code>r</code></span>이 되고, $S$에서 $O$를 제외한 부분의 접미사이면서 $Y$의 접두사가 될 수 있는 가장 긴 문자열은 <span style="color:#e74c3c;"><code>t</code></span>가 된다. 따라서 $T$, $S$, $Y$는 각각 <span style="color:#e74c3c;"><code>toyca</code></span>, <span style="color:#e74c3c;"><code>rust</code></span>, <span style="color:#e74c3c;"><code>oon</code></span>이 되고, <strong>계정 이름의 후보</strong>는 <span style="color:#e74c3c;"><code>toycarustoon</code></span>이 된다.</li>
</ul>
</li>
</ol>
</li>
</ul>

<p style="text-align: center;"><img alt="" src="https://upload.acmicpc.net/2c4622ea-051c-4ccf-b745-8ef7c6e76e21/-/preview/" style="width: 450px; height: 314px;"></p>

<p>만약 <strong>계정 이름의 후보</strong>의 길이가 20을 넘는다면 계정 이름의 길이 제한에 걸리기 때문에 <span style="color:#e74c3c;"><code>toycartoon</code></span>을 계정 이름으로 사용해야 한다. 아니라면 <strong>계정 이름의 후보</strong>를 새로운 계정 이름으로 사용한다. 토카를 도와 토카가 사용할 새로운 계정의 이름을 구해보자!</p>

## 입력 

 <p>첫째 줄에 토카가 공부하고자 하는 언어의 이름 $S$가 주어진다. $S$는 알파벳 소문자로 이루어져 있으며, 길이는 $1$ 이상 $20$ 이하이다.</p>

## 출력 

 <p>첫째 줄에 토카가 사용할 계정의 이름을 출력한다.</p>

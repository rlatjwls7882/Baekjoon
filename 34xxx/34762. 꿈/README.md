# [Silver III] 꿈

[문제 링크](https://www.acmicpc.net/problem/34762) 

## 문제 설명

<p>쿠는 $1$번째 날부터 $N$번째 날까지 매일 하나의 꿈을 꾼다. 하지만 악몽을 꾸면 충격을 받아서 최근에 꾼 꿈들을 잊어버린다.</p>

<p>$i$번째 날에 악몽을 꾸면, $K$일 전에 꾼 꿈부터 $i$번째 날에 꾼 꿈까지 모두 잊어버린다. 구체적으로 $\max(1, i-K), \cdots , i-1, i$번째 날에 꾼 꿈들을 모두 잊어버린다. 이 중 이미 잊은 꿈이 있다면, 해당 꿈은 그대로 잊은 상태가 유지된다.</p>

<p>집합 $S = \{S_1, S_2, ... , S_M \}$이 주어진다. $N$번째 날에 꿈꾼 직후, 쿠가 $S_i$번째 날에 꾼 꿈들만 기억할 수 있는지 판별해 보자.</p>

## 입력 

 <p>첫째 줄에 꿈을 꾼 날의 수, 악몽이 영향을 미치는 기간을 의미하는 두 정수 $N$, $K$가 공백으로 구분되어 주어진다. $(1\leq K \leq N \leq 200\, 000)$</p>

<p>둘째 줄에 집합 $S$의 크기를 의미하는 정수 $M$이 주어진다. $(1\leq M \leq N)$</p>

<p>셋째 줄에 정수 $S_1, S_2, \cdots , S_M$이 공백으로 구분되어 주어진다. $(1\leq S_1 <S_2<\cdots <S_M\leq N)$</p>

## 출력 

 <p>집합 $S$에 대해 $S_i$번째 날에 꾼 꿈들만 기억할 수 있다면 <span style="color:#e74c3c;"><code>YES</code></span>를, 그렇지 않다면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>

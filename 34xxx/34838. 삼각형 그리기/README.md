# [Gold V] 삼각형 그리기

[문제 링크](https://www.acmicpc.net/problem/34838) 

## 문제 설명

<p>$N$개의 점이 원 위에 일정한 간격으로 놓여 있다. 이 중 $3$개의 점을 골라 이어서 만들 수 있는 삼각형 중 서로 다른 삼각형의 개수를 구하여라.  </p>

<p>회전했을 때 모양이 일치하는 삼각형은 같은 삼각형이다.</p>

## 입력 

 <p>첫째 줄에 정수 $N$이 주어진다. ($3\leq N \leq 100\,000$)</p>

## 출력 

 <p>첫째 줄에 삼각형의 개수를 출력한다.</p>

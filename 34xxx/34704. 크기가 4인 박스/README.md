# [Silver V] 크기가 4인 박스

[문제 링크](https://www.acmicpc.net/problem/34704) 

## 문제 설명

<p>탐욕스러운 동건이는 $N$개의 물건을 훔쳤고, 이제 모든 물건들을 크기가 $4$인 박스들에 나누어 담아야 한다. 각 물건의 크기는 $1$ 이상 $4$ 이하의 정수이며, 하나의 박스에는 크기의 합이 $4$를 넘지 않도록 여러 물건을 넣을 수 있다. 동건이가 모든 물건을 담기 위해 필요한 박스의 최소 개수를 구하여라.</p>

## 입력 

 <p>첫째 줄에 물건의 개수를 나타내는 정수 $N$이 주어진다. $(1 \le N \le 100\,000)$</p>

<p>둘째 줄에는 각 물건의 크기를 나타내는 $N$개의 정수 $a_i$가 주어진다. $(1 \le a_i \le 4)$</p>

## 출력 

 <p>동건이가 모든 물건을 담기 위해 필요한 박스의 최소 개수를 출력한다.</p>

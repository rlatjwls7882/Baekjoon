# [Bronze II] Dviračių lenktynės

[문제 링크](https://www.acmicpc.net/problem/30077) 

## 문제 설명

<p>Bitų karalystėje vyksta dviračių treko lenktynės. Visi dviratininkai startuoja vienu metu ir važiuoja $N$ ratų. Jei kurį lenktynininką ratu aplenkia kitas, aplenktasis toliau varžybų nebetęsia. Jei sportininkas, kirsdamas finišo liniją, aplenkia varžovą ratu, aplenktasis taip pat iškrenta iš varžybų.</p>

<p>Žinodami, per kiek laiko kiekvienas dviratininkas įveikia vieną treko ratą bei kiek ratų trunka lenktynės, raskite, kiek dalyvių nebus aplenkti ratu ir finišuos varžybose.</p>

## 입력 

 <p>Pirmojoje eilutėje pateikti trys natūralieji skaičiai atskirti tarpu:</p>

<ul>
	<li>$N$ – lenktynininkų skaičius;</li>
	<li>$M$ – varžybų trukmė ratais;</li>
	<li>$L$ – treko rato ilgis metrais.</li>
</ul>

<p>Kitose $N$ eilučių pateikta, per kiek sekundžių ($T_i$) kiekvienas iš sportininkų įveikia vieną treko ratą. $T_i$ - natūralusis.</p>

## 출력 

 <p>Išveskite sportininkų, kurie nebuvo aplenkti ratu ir baigė varžybas, skaičių.</p>

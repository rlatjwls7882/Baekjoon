# [Silver V] $02493$

[문제 링크](https://www.acmicpc.net/problem/35146) 

## 문제 설명

<p>$02493$. 누군가에게는 아무 의미가 없을 수 있지만, PC를 풀고 온 여러분에게는 $39420$을 뒤집은 수이다.</p>

<p>앞선 <a href="/problem/35145">PC : 39420</a>문제의 데이터를 만들던 동우는 큰 난관에 봉착했다. 이 문제의 답이 최대가 되는 데이터를 알기 어려웠던 동우는, 대신 다음과 같이 $2$차원 배열 $A$를 만들어 데이터에 추가했다.</p>

<ul>
<li>먼저 행과 열의 개수 $N$과 $M$을 정한다.</li>
<li>다음으로 정수 상수 $a$와 $b$를 정한다.</li>
<li>이후 $i$행 $j$열의 원소 $A_{ij}$를 $\left( ai+bj \right)\bmod 10$으로 한다.</li>
</ul>

<p><a href="/problem/35145">원래 문제</a>는 여기서 그릴 수 있는 $\binom{N+1}{2}\times\binom{M+1}{2}$개의 직사각형들 중 직사각형 안의 숫자가 모두 다른 직사각형이 몇 개인지 구하는 것이었다. 반대로 동우는 원래 문제의 답이 $X$개가 되도록 $N,M,a,b$를 정하고자 한다. 동우를 도와 $N,M,a,b$를 정해 보자.</p>

## 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T(1\le T\le 10^6)$가 주어진다.</p>

<p>두 번째 줄에 각 테스트 케이스의 동우의 목표 정수 $X$를 의미하는 서로 다른 $T$개의 정수 $X_1,X_2,\cdots ,X_T(1\le X_i\le 10^6)$가 공백으로 구분되어 오름차순으로 주어진다.</p>

## 출력 

 <p>첫 번째 줄부터 $T$줄에 걸쳐 각 테스트 케이스별로 한 줄씩 가능한 정수 쌍 $N,M,a,b(1\le N,M,a,b\le 10^{6})$를 공백으로 구분하여 출력한다.</p>

<p>가능한 답이 여러 개라면 아무거나 하나 출력하며, 불가능하면 대신 <span style="color:#e74c3c;"><code>-1 -1 -1 -1</code></span>을 출력한다.</p>

# [Bronze III] 지금부터 완장 찬양을 시작하겠습니다

[문제 링크](https://www.acmicpc.net/problem/35262)

## 문제 설명

<p>완장이 제3회 피갤컵 개최를 공지하자 피붕이는 완장을 찬양하기 시작했다.</p>

<p>이러한 성원에 보답하고자 완장은 찬양해 준 피붕이의 지난 $N$일간 피갤 방문 기록을 가져와 아래와 같은 기준에 해당되지 않는다면 햄부기를 주기로 한다.</p>

<ul>
<li>연속으로 방문하지 않은 기간이 $K$일 이상인 경우가 있다면, 완장은 괘씸하다 생각하여 햄부기를 주지 않는다.</li>
</ul>

<p>위 기준을 바탕으로 피붕이가 햄부기를 받을 수 있는지 알아보자.</p>

## 입력

<p>첫째 줄에 $N$과 $K$가 공백으로 구분되어 주어진다. $(1 \le K \le N \le 1\,000\,000)$</p>

<p>둘째 줄에는 피붕이의 지난 $N$일간 피갤 방문 기록을 나타내는 길이 $N$의 이진 수열이 주어진다. 각 원소는 해당 날짜에 방문했으면 $1$, 방문하지 않았으면 $0$을 나타낸다.</p>

## 출력

<p>피붕이가 완장한테 햄부기를 받을 수 있으면 $1$, 아니면 $0$을 출력한다.</p>

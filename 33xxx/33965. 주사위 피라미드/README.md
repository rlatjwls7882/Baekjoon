# [Bronze I] 주사위 피라미드

[문제 링크](https://www.acmicpc.net/problem/33965) 

## 문제 설명

<p>2025년을 맞이하여 SCSC에는 예술 주사위 행사가 열렸다. 행사에서는 주사위를 면끼리 맞닿게 쌓아 가장 멋진 모양을 만드는 사람을 가리는 게임을 진행한다. 주사위는 모두 크기가 같은 정육면체 모양으로 다음과 같은 전개도를 가지는 주사위를 사용한다.</p>

<p style="text-align: center;"><img alt="" src="" style="width: 300px; height: 224px;"></p>

<p>종환이는 호기롭게 나서서 피라미드 모양으로 쌓은 작품을 선보였다.</p>

<p style="text-align: center;"><img alt="" src="" style="width: 300px; height: 279px;"><img alt="" src="" style="height: 279px; width: 345px;"></p>

<p>처음부터 나타난 강적에 모두가 박수를 치고 있었지만 성재만은 달랐다. 성재는 주사위 피라미드를 $N$층으로 쌓았을 때 바닥에 닿은 면과 주사위끼리 맞닿은 면을 제외한, 겉으로 보이는 모든 면에 그려진 눈의 합이 가질 수 있는 최댓값과 최솟값이 궁금했다. $N$층 주사위 피라미드는 $N$층에는 $1$개, $N-1$층에는 $3$개, $N-2$층에는 $6$개, $\cdots$, $1$층에는 $\displaystyle\frac{N(N+1)}{2}$개의 주사위가 위 그림과 같은 규칙으로 쌓인 모양을 말한다.</p>

<p>계산을 끝낸 성재는 답이 하나인 것을 선호했기 때문에 둘을 더한 값을 나직이 말한 후 홀연히 행사장을 떠났다. 성재가 말한 값은 무엇이었을까?</p>

## 입력 

 <p>첫째 줄에 주사위 피라미드의 층수를 나타내는 정수 $N$이 주어진다. $(1 \leq N \leq 2 \, 025)$</p>

## 출력 

 <p>첫째 줄에 $N$층 주사위 피라미드를 쌓았을 때, 겉으로 보이는 눈의 합이 가질 수 있는 최댓값과 최솟값을 더한 값을 출력한다.</p>

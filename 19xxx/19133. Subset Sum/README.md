# [Diamond V] Subset Sum

[문제 링크](https://www.acmicpc.net/problem/19133) 

## 문제 설명

<p>Given a multiset of integers $A = \{a_1, a_2, \dots, a_n\}$, print the least $k$ sums among all non-empty subsets in sorted order.</p>

## 입력 

 <p>The first line contains $2$ integers $n, k$ ($1 \leq n \leq 200000, 1 \leq k \leq \min\{2^n - 1, 200000\}$).</p>

<p>The second line contains $n$ integers $a_1, a_2, \dots, a_n$ ($|a_i| \leq 10^9$).</p>

## 출력 

 <p>$k$ integers denote the least $k$ sums.</p>

# [Diamond IV] mod와 쿼리

[문제 링크](https://www.acmicpc.net/problem/24547) 

## 문제 설명

<p>길이가 $N$인 양의 정수들의 수열 $A_1, A_2, ... A_N$이 주어진다. 이때, 다음과 같은 쿼리를 수행하는 프로그램을 작성하시오. </p>

<ul>
	<li>$1$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum_{i=1}^N {({A_i}\bmod{X})} $를 출력한다.</li>
	<li>$2$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum_{i=1}^N {({X}\bmod{A_i})} $를 출력한다.</li>
	<li>$3$ $i$ $X$ : $A_i$의 값을 양의 정수 $X$로 바꾼다. ($1\le i \le N$) </li>
</ul>

<p>${P}\bmod{Q}$는 $P$를 $Q$로 나눈 나머지를 뜻한다. </p>

## 입력 

 <p>첫째 줄에 수열의 길이 $N$이 주어진다. ($1\le N \le 100\,000$)</p>

<p>둘째 줄에는 양의 정수들의 수열 $A_1, A_2, ... A_N$이 주어진다. ($1\le A_i \le 100\,000$)</p>

<p>셋째 줄에는 쿼리의 개수 $Q$가 주어진다. ($1\le Q \le 100\,000$)</p>

<p>넷째 줄부터 $Q$개의 줄에 걸쳐 쿼리가 주어진다. ($1\le X \le 100\,000$) </p>

## 출력 

 <p>1, 2번 쿼리에 대해서 정답을 한 줄에 하나씩 출력한다. </p>

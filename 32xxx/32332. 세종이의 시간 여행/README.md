# [Bronze I] 세종이의 시간 여행

[문제 링크](https://www.acmicpc.net/problem/32332) 

## 문제 설명

<p>세종대학교에 입학한 세종이에게는 어릴 적부터 시간 여행이라는 큰 꿈이 있었다. 학교를 열심히 다니던 세종이는 얼마 전 한양대학교에 다니던 친구 한양이에게 타임머신에 대한 소문 하나를 전해 듣게 되었다. 바로 한양대학교에서 비밀리에 타임머신을 개발했다는 소문이었다!</p>

<p>타임머신이 있는 한양대학교 연구실로 달려간 세종이는 아래의 내용이 적힌 타임머신 사용설명서를 발견하였다.</p>

<ul>
	<li>이동하려는 연도와 월, 일을 입력한다.</li>
	<li>이동하려는 위도와 경도를 소수점 셋째 자리까지 정확하게 입력한다.</li>
	<li>이동 버튼을 누른다.</li>
</ul>

<p>사용설명서를 읽고서 그 즉시 시간 여행을 하려던 세종이는 미처 읽지 않은 의문의 쪽지를 발견하였다!</p>

<p>의문의 쪽지에는 다음과 같이 적혀있었다.</p>

<ul>
	<li>타임머신이 아직 미완성된 상태이므로 입력한 시공간 좌표의 반대 방향으로 이동하게 된다는 치명적인 버그가 존재한다.</li>
	<li>시간의 경우, 타임머신에 입력한 시간이 현재보다 $X$일 후라면 $X$일 전으로 이동하며, $X$일 전이라면 $X$일 후로 이동한다.</li>
	<li>공간에서 위도의 경우, 타임머신에 입력한 위도가 현재 위치의 위도보다 북쪽이라면(증가하는 방향이라면) 실제로는 같은 거리만큼 남쪽으로(감소하는 방향으로) 이동하며, 현재 위치의 위도보다 남쪽이라면(감소하는 방향이라면) 실제로는 같은 거리만큼 북쪽으로(증가하는 방향으로) 이동한다.
	<ul>
		<li>모든 위도의 값은 대한민국 근처로 주어지므로, 이동 후 위도가 $-90$도 미만이 되거나 $90$도를 초과하는 경우는 고려하지 않아도 된다.</li>
		<li><strong>문제의 모든 입력과 출력에서 위도가 $36$도 이상 $39$도 이하임이 보장된다.</strong></li>
	</ul>
	</li>
	<li>공간에서 경도의 경우, 타임머신에 입력한 경도가 현재 위치의 경도보다 동쪽이라면(증가하는 방향이라면) 실제로는 같은 거리만큼 서쪽으로(감소하는 방향으로) 이동하며, 현재 위치의 경도보다 서쪽이라면(감소하는 방향이라면) 실제로는 같은 거리만큼 동쪽으로(증가하는 방향으로) 이동한다.
	<ul>
		<li>모든 경도의 값은 대한민국 근처로 주어지므로, 이동 후 경도가 $-180$도 미만이 되거나 $180$도를 초과하는 경우는 고려하지 않아도 된다.</li>
		<li><strong>문제의 모든 입력과 출력에서 경도가 $116$도 이상 $140$도 이하임이 보장된다.</strong></li>
	</ul>
	</li>
	<li>예를 들어, 위도 $37.0$도, 경도 $127.0$도에서 출발해 위도 $37.5$도, 경도 $130.0$도를 타임머신에 입력하면, 실제로는 위도 $36.5$도, 경도 $124.0$도로 이동한다.</li>
</ul>

<p>세종이가 타임머신을 탑승한 시공간 좌표와 타임머신을 타고 이동하려는 시공간 좌표가 주어질 때, 타임머신에 입력해야 하는 시공간 좌표를 계산해 보자.</p>

<p><strong>계산의 편의를 위해 $1$년을 $12$개월로, $1$개월을 $30$일로 가정한다.</strong></p>

## 입력 

 <p>첫 번째 줄에 세종이가 타임머신에 탑승한 현재 시점의 연도 $Y_0$, 월 $M_0$, 일 $D_0$, 위도 $T_0$, 경도 $P_0$이 공백으로 구분되어 주어진다. $(2000 \le Y_0 \le 3000;$ $1 \le M_0 \le 12;$ $1 \le D_0 \le 30;$ $37 \le T_0 \le 38;$ $124 \le P_0 \le 132)$</p>

<p>두 번째 줄에 세종이가 타임머신을 타고 이동하려는 시점의 연도 $Y_1$, 월 $M_1$, 일 $D_1$, 위도 $T_1$, 경도 $P_1$이 공백으로 구분되어 주어진다. $(2000 \le Y_1 \le 3000;$ $1 \le M_1 \le 12;$ $1 \le D_1 \le 30;$ $37 \le T_1 \le 38;$ $124 \le P_1 \le 132)$</p>

<p>모든 연도와 월, 일은 정수로 주어지고, 모든 위도와 경도는 정확히 소수점 셋째 자리까지 주어진다.</p>

<p>모든 수 앞에 불필요한 $0$이 붙지 않는다.</p>

## 출력 

 <p>세종이가 타임머신에 입력해야 하는 시공간 좌표의 연도, 월, 일, 위도, 경도를 하나의 줄에 공백으로 구분하여 출력한다.</p>

<p>연도와 월, 일은 정수로, 위도와 경도는 정확히 소수점 셋째 자리까지 출력해야 한다.</p>

<p>모든 수 앞에 불필요한 $0$이 붙어서는 안 된다.</p>
